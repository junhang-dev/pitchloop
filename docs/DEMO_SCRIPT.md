# Demo Script (2-3 minutes)

1) Sign in
2) Create a project (title, goal, audience, duration)
3) Create a session
4) Upload rehearsal media
5) Run analysis
6) Show results:
   - speaking rate
   - fillers
   - pauses (estimated ok)
   - next actions checklist
7) Return to project page:
   - session list
   - simple trend for speakingRateWpm
